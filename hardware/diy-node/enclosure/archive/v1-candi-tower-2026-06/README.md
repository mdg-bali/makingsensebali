# Archived: v1 "candi tower" functional enclosure — June 2026

First functional build (ICD v1.3). Single tapered candi/meru tower body +
5-roof crest + bottom bayonet cap. Generator: `enclosure_plus_v1.py`.

Reached: watertight, interference-clean, support-free, sliced
(~89 g / ~4 h for one Plus). Validated the internal architecture
(vertical HM3301 in rails, perfboard on back standoffs, keyholes, bottom
bayonet, chimney) and proved the packaging math (80 mm carrier forces a
tall body; no LiPo fits beside it → battery deferred to a future cap).

Superseded 2026-06-08 by **v2**, when Tomas reframed the brief:
functionality is the form-giver, not the cultural concept; keep the
batik/fractal lineage as surface language; use triangulated/faceted
surfaces (digital semi-organic); airflow + water-exclusion paramount;
minimise material + print time; modular.

What v2 keeps from here: the internal architecture and all the verified
component dimensions and clearances. What changes: monolithic tower →
modular spine + slide-on faceted shell + shared caps; smooth revolves →
faceted octagonal louver geometry.
